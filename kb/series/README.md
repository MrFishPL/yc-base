# kb/series/

**What goes here:** one landing page per show / playlist / series — `<slug>.md`, `type: series`
(e.g. `startup-school`, `lightcone`, `how-to-start-a-startup`, `dalton-and-michael`). Describes
the series and links to its talk pages. Useful as a tier in `kb/INDEX.md`.

**What does NOT go here:** individual talk content (→ `kb/talks/`).
