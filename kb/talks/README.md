# kb/talks/

**What goes here:** one page per YouTube video — `<slug>.md`, `type: yc-video`.
Holds *that talk's* thesis + 5–12 timestamped key moments, and `[[wikilinks]]` up to the topic
and entity pages. `resource:` (the YouTube URL) is required — it's the citation anchor.

**What does NOT go here:** cross-cutting synthesis (→ `kb/topics/`), person/company bios
(→ `kb/entities/`), or raw/cleaned transcripts (→ `transcripts/`). Don't restate shared advice
here — cite the topic page instead.
