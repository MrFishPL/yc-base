# AI Is Unlocking Millions Of New Builders — Compiled Truth

Emergent's twin founders describe pivoting from an enterprise coding agent into a platform that lets non-technical domain experts ship production-ready software, arguing AI is expanding who can start a business rather than just eliminating jobs.

**Key ideas (with timestamps):**
- Emergent pivoted from an automated software-testing tool to a general coding agent after realizing that solving verification is the key loop that lets agents keep running productively [ai-is-unlocking-millions-of-new-builders @ 02:19](https://youtu.be/8SVocWnDHwE?t=139)
- Emergent built its own coding agent and became #1 on the SWE-bench benchmark within two months, discovering multi-agent orchestration and agent memory before those ideas appeared in published research [ai-is-unlocking-millions-of-new-builders @ 03:08](https://youtu.be/8SVocWnDHwE?t=188)
- After first trying an enterprise go-to-market, Emergent pivoted to non-technical users and found 80% of its users have zero programming knowledge [ai-is-unlocking-millions-of-new-builders @ 04:22](https://youtu.be/8SVocWnDHwE?t=262)
- Entering the market after Lovable and Bolt let Emergent learn from what wasn't working and rebuild the platform end-to-end for production-ready apps (code review, testing, deployment, security, hosting) instead of just front-end prototyping [ai-is-unlocking-millions-of-new-builders @ 06:00](https://youtu.be/8SVocWnDHwE?t=360)
- Emergent runs its own Kubernetes-based cloud sandbox infrastructure, rather than a third-party sandbox, so agents get identical, fast feedback during both build and deploy, reducing production failures [ai-is-unlocking-millions-of-new-builders @ 09:47](https://youtu.be/8SVocWnDHwE?t=587)
- Early distribution came from building a large influencer network on TikTok and Instagram, targeted at users who wanted to ship "real" apps rather than prototypes [ai-is-unlocking-millions-of-new-builders @ 07:54](https://youtu.be/8SVocWnDHwE?t=474)
- Emergent gives its agents long-term memory across sessions by distilling past task trajectories into reusable "skills" run through a CI/CD process, since agent-generated skills underperform [ai-is-unlocking-millions-of-new-builders @ 11:47](https://youtu.be/8SVocWnDHwE?t=707)
- Because non-technical users get intimidated by seeing code diffs or JSON, Emergent deliberately hides its powerful internal coding-agent tooling (used internally as a Claude Code replacement) behind a simplified interface [ai-is-unlocking-millions-of-new-builders @ 13:09](https://youtu.be/8SVocWnDHwE?t=789)
- The founders see the software market as expanding, not shrinking: non-developers are becoming developers, and PM/designer/engineer roles are collapsing into single people using AI tools [ai-is-unlocking-millions-of-new-builders @ 17:39](https://youtu.be/8SVocWnDHwE?t=1059)
