# Aravind Srinivas: Perplexity's Race to Build Agentic Search — Compiled Truth

Perplexity's CEO argues that speed is the company's only real moat against well-funded copycats, and that Perplexity's next bet — an agentic browser called Comet — is deliberately hard to replicate quickly.

- Perplexity's next big bet is the Comet browser, reframed as a "cognitive operating system" — an assistant that can run parallel, asynchronous agentic tasks across your tabs, email, and accounts [aravind-srinivas-perplexity-s-race-to-build-agentic-search @ 01:02](https://youtu.be/2jOnoTEk-xA?t=62)
