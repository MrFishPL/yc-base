# François Chollet: Why Scaling Alone Isn't Enough for AGI — Compiled Truth

François Chollet argues that deep learning's scaling paradigm will get us far but is not optimal, and that his new lab Indium is betting on symbolic program synthesis as an alternative path to AGI, while sharing lessons on pursuing contrarian bets and building durable open-source communities.

**Key ideas (with timestamps):**
- Indium is building a new branch of machine learning that replaces parametric curves and gradient descent with small symbolic models found via "symbolic descent," aiming to be far more data- and compute-efficient than deep learning [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 01:46](https://youtu.be/k2ZLQC8P7dc?t=106)
- His advice for contrarian bets: if an idea has a low chance of success but would be big if it worked and no one else is pursuing it, it's worth trying — comparing this directly to the YC mission [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 05:22](https://youtu.be/k2ZLQC8P7dc?t=322)
- Coding agents succeeded so fast because code provides a verifiable reward signal, letting any domain with formally checkable outputs be automated with current LLM technology [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 06:29](https://youtu.be/k2ZLQC8P7dc?t=389)
- He distinguishes "AGI as automating economically valuable tasks" from his own definition of general intelligence as human-level skill-acquisition efficiency, and expects the former to arrive first [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 10:02](https://youtu.be/k2ZLQC8P7dc?t=602)
- The ARC-AGI benchmark series is deliberately a moving target that keeps re-measuring the residual capability gap between frontier AI and humans rather than a fixed pass/fail test [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 44:19](https://youtu.be/k2ZLQC8P7dc?t=2659)
- He argues compute is "the great equalizer" — pouring the same investment into almost any alternative approach (e.g. genetic algorithms) would likely also produce exciting results, so more people should explore neglected paths [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 46:39](https://youtu.be/k2ZLQC8P7dc?t=2799)
- When evaluating new AI research approaches, the key criterion is whether capability improvements can scale without requiring constant human engineering effort in the loop [fran-ois-chollet-why-scaling-alone-isn-t-enough-for-agi @ 50:11](https://youtu.be/k2ZLQC8P7dc?t=3011)
