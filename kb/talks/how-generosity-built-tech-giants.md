# How Generosity Built Tech Giants — Compiled Truth

Michael Seibel and Dalton Caldwell argue that lasting tech businesses are built by giving customers more value than they take, using Google, Microsoft, and the free software movement as evidence.

**Key ideas (with timestamps):**
- The basic idea underpinning technological progress: if you sell someone a tool that lets them make more money than the tool costs, you create real value, as seen with early Google Ads and Excel making office workers dramatically more productive. [how-generosity-built-tech-giants @ 00:20](https://youtu.be/RD5EAMOh7fE?t=20)
- Many founders build products that deliver minimal value to users because they're chasing growth metrics to raise money rather than solving a real problem. [how-generosity-built-tech-giants @ 01:23](https://youtu.be/RD5EAMOh7fE?t=83)
- Fear of "being a consultant" stops founders from digging into individual customers' real problems, but early on that hands-on consulting-like work is exactly how you learn what's generalizable. [how-generosity-built-tech-giants @ 02:40](https://youtu.be/RD5EAMOh7fE?t=160)
- Google's founders weren't performance-marketing experts when they started; understanding how customers like mortgage-refinance advertisers actually made money came only through building and iterating with real users. [how-generosity-built-tech-giants @ 03:54](https://youtu.be/RD5EAMOh7fE?t=234)
- Offering value upfront in cold outreach — like recording and critiquing a prospect's onboarding flow before asking for anything — gets far better responses than requesting time for nothing in return. [how-generosity-built-tech-giants @ 06:35](https://youtu.be/RD5EAMOh7fE?t=395)
- The history of computing is built on giving away enormous value for free — Linux, GCC, SQLite, and the free software movement — long before proprietary software business models existed. [how-generosity-built-tech-giants @ 08:12](https://youtu.be/RD5EAMOh7fE?t=492)
- Giving more value than you take, while still genuinely solving the customer's problem, is what creates outsized economic value and can produce wild startup success. [how-generosity-built-tech-giants @ 09:34](https://youtu.be/RD5EAMOh7fE?t=574)
