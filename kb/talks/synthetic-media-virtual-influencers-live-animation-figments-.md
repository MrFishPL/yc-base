# Synthetic Media: Virtual Influencers & Live Animation: Figments (S19) - YC Gaming Tech Talks 2020 — Compiled Truth

Figments co-founder Jay Rosenkrantz describes building a "WWE for esports" using virtual characters with fictional storylines who compete in real games on Twitch, powered by custom virtual production and motion capture technology.

**Key ideas (with timestamps):**
- Figments pitches itself as building the next version of the WWE for esports, creating brand-new virtual characters with fictional storylines who play esports competitively. [synthetic-media-virtual-influencers-live-animation-figments- @ 00:28](https://youtu.be/1Rq0ltksFEg?t=28)
- Their first character, Morales, is a robot built to dominate Super Smash Brothers Ultimate and grew into a partner streamer on Twitch with almost 10,000 YouTube subscribers via a viral event called "Beat the Robot." [synthetic-media-virtual-influencers-live-animation-figments- @ 00:56](https://youtu.be/1Rq0ltksFEg?t=56)
- To create characters like Morales, the team built its own virtual production tooling that mixes live streaming video/audio pipelines with motion capture front-ends. [synthetic-media-virtual-influencers-live-animation-figments- @ 02:28](https://youtu.be/1Rq0ltksFEg?t=148)
- COVID forced the originally LA-based, in-house studio team to go fully remote, which turned out to be a blessing since it let them collaborate with talent anywhere with a high-speed internet connection. [synthetic-media-virtual-influencers-live-animation-figments- @ 02:58](https://youtu.be/1Rq0ltksFEg?t=178)
- The tooling was built to be performer-friendly and scalable so talent can create character content from a home office or streaming room rather than a large motion-capture stage. [synthetic-media-virtual-influencers-live-animation-figments- @ 03:21](https://youtu.be/1Rq0ltksFEg?t=201)
- The technology relies on Unreal Engine and ARKit on iPhone for real-time game engine-driven live animation and mocap, similar to techniques used on The Mandalorian. [synthetic-media-virtual-influencers-live-animation-figments- @ 04:31](https://youtu.be/1Rq0ltksFEg?t=271)

- That Reebok partnership led other brands to get excited, resulting in a further partnership with Guayaki Yerba Mate. [synthetic-media-virtual-influencers-live-animation-figments- @ 06:38](https://youtu.be/1Rq0ltksFEg?t=398)
